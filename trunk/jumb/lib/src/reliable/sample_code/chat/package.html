Source code for the sample chat application.

This application uses reliable multicast for a multi-sender chat application. The application presents a window in which are displayed messages from all other users. Users add messages by typing them into a separate panel.
Command line arguments are as follows:

java com.sun.multicast.reliable.applications.chat.Chat [-transport transport-name] addr port userName
    transport-name - the name of the transport protocol. The default transport protocol is LRMP.
    addr - multicast address to use
    port - port number to use
    userName - a name to identify the user on the Chat screen
